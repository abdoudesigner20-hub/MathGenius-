
### بعد التوليد:
- حمّل ملف .jks واحفظه بأمان
- حوّله لـ Base64 عبر: https://base64.guru/converter/encode/file

## 🐙 الخطوة 2 — رفع الملفات على GitHub

### من الهاتف:
1. افتح github.com وسجّل دخولك
2. أنشئ Repository جديد: **MathGenius** (Public)
3. أنشئ كل ملف يدوياً:
   - اضغط **Add file → Create new file**
   - اكتب المسار الكامل مثل: `app/src/main/java/com/mathgenius/app/MainActivity.java`
   - الصق محتوى الملف
   - اضغط **Commit changes**

### ⚠️ مهم جداً:
- ارفع ملف HTML باسم `index.html` في المسار:
  `app/src/main/assets/index.html`

---

## 🔒 الخطوة 3 — إضافة Secrets في GitHub

```
Repository → Settings → Secrets and variables → Actions
→ New repository secret
```

أضف هذه الـ Secrets:

| الاسم | القيمة |
|-------|--------|
| KEYSTORE_BASE64 | (الـ Base64 الذي حصلت عليه من الخطوة 1) |
| KEYSTORE_PASSWORD | MathGenius@2024 |
| KEY_ALIAS | mathgenius |
| KEY_PASSWORD | MathGenius@2024 |
